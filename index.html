Conhecimentos

HTML

HTML é a linguagem de marcação responsável por todo o conteúdo presente nas páginas da internet. É através dele que podemos definir os elementos que se farão visíveis ao acessar determinado site.

CSS

A linguagem CSS é a linguagem responsável por todo estilo visível nas páginas web. Dessa forma o CSS se constitui uma ferramenta indispensável para o desenvolvedor Front End afim de trazer identidade visual ao projeto a ser entregue.

Design

Um bom desenvolvedor web deve deter também conhecimento prévio em design, tendo em vista que o planejamento visual do estilo será o primeiro passo para um bom projeto.

Javascript

O desenvolvedor deve ter o pleno conhecimento da linguagem de programação JavaScript bem como ECMAScript, possibilitando dessa forma a criação de páginas web com interação plena e didática com o usuário.

Boostrap

O Boostrap é uma ferramenta que auxilia o trabalho do desenvolvedor no quesito da responsividade dos projetos uma vez que são esperados usuários de plataformas Mobile e Desktop.

Git e GitHub

O Git e GitHub são plataformas de integração e compartilhamentos de repositórios, através deles é possível acessarmos e até mesmo modificarmos códigos de projetos públicos.
